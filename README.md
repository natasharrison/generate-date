# Generate-Date You + Me

## Description

Dinner date web application that randomly generates a dinner recipe, a cocktail recipe and a movie or music playlist suggestion to help take the decision-making out of date night.

## Built With

- HTML
- CSS
- JavaScript
- Recipe API
- Cocktail API
- Movie/Music API
